📌 MÉTODO MÁS SIMPLE (RECOMENDADO)
══════════════════════════════════════════════════════════════

1. Abre una terminal en la carpeta del proyecto
2. Ejecuta:

   python main.py

   ¡Listo! El juego debería iniciar.

══════════════════════════════════════════════════════════════
⚡ SI ES TU PRIMERA VEZ
══════════════════════════════════════════════════════════════

Si no tienes las dependencias instaladas:

   pip install pygame

Luego ejecuta:

   python main.py

══════════════════════════════════════════════════════════════
🎮 CONTROLES DEL JUEGO
══════════════════════════════════════════════════════════════

Durante la INTRO:
  [ESPACIO]  - Saltar animación / Comenzar juego

Durante el JUEGO:
  [M]        - Minar materiales (costo: 2 oxígeno)
  [R]        - Reparar nave (costo: 3 oxígeno + materiales)
  [I]        - Ver inventario
  [D]        - Ver deudas activas
  [P]        - Ver progreso de reparación
  [ESPACIO]  - Avanzar diálogos
  [ESC]      - Cerrar paneles / Salir
